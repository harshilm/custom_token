
WHAT IS THIS MODULE FOR? 
	You may need it when you want to: - Export and/or import custom tokens via features - Create custom token for your site and take those configurations via features from one environment to another 

USAGE INSTRUCTIONS
	EXPORT 
		1. Create your custom token from Drupal admin. 
		2. Go to the create features link (admin/structure/features)
		3. On the right side you will be able to see field set name as Custom token features 
		4. Select the list of token and export as features 
 
	IMPORT
		It will work same as other features module
